|---------|------------|----------|-----------|-----------|

现货交易中,Maker(挂单)和Taker(吃单)是两个概念。简单说,你在市场上挂一个限价单等别人来买,你就是Maker;如果你直接按市场价成交,你就是Taker。因为Maker能增加市场深度,所以手续费通常更便宜。
